Technical Highlights

Built with industry-standard best practices.

Security First

Implemented strict JWT Bearer token authentication with Role-Based Access Control (RBAC). Every API endpoint is protected to ensure only authorized users can access sensitive data.

Scalable Storage

Integrated robust file upload systems (Multer) handling agent verification images, pickup photos, and banner assets securely. Structured directory management for clean asset organization.

Atomic Transactions

Financial operations (Pickups, Wallet Credits) utilize MongoDB Sessions to ensure data integrity. Money is never lost or double-counted during operations.

API Endpoints Reference

A glimpse into the 40+ endpoints powering the system.

Authentication

  • POST /auth/send-otp - Login/Register via OTP
  • POST /auth/verify-otp - Verify & Get Token
  • GET /auth/google - OAuth 2.0 Integration
  • GET /auth/me - Get Profile

Customer Module

  • POST /customer/pickups - Book Pickup
  • GET /customer/pickups - History & Status
  • GET /customer/wallet - Balance Check
  • GET /customer/pickups/rates - Live Scrap Rates

Agent Module

  • GET /agent/dashboard - Daily Stats
  • PUT /agent/pickups/:id/accept - Claim Job
  • POST /agent/location - GPS Stream
  • PUT /agent/pickups/:id/log-items - Upload Photos/Weights
  • POST /agent/payout/request - Withdraw Earnings

Admin Controls

  • GET /admin/dashboard - System Analytics
  • POST /admin/scrap-items - Manage Pricing
  • PUT /admin/agents/:id/approve - Verify Agents
  • POST /admin/banners - Content Management
  • GET /admin/payouts - Financial Oversight
